Ứng dụng của HTML
Thông tin thêm

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu được sử dụng để tạo cấu trúc cho trang web. Nó kết hợp với CSS và JavaScript để xây dựng giao diện và tính năng.

Nội dung học

  • HTML Cơ bản
    • Thẻ tiêu đề (h1-h6)
    • Đoạn văn (p)
    • Danh sách (ul, ol, li)
  • CSS Cơ bản
    • Chọn phần tử (selectors)
    • Thuộc tính màu sắc
    • Font chữ
  • JavaScript Cơ bản
